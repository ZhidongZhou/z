MacOS Dock Optimization 程序坞优化

相·遇
15-05-2020
执迷终端
MacOS Tips

1. finder中显示当前路径:

首先打开 Terminal

关闭路径显示， 只将 YES 改为 NO。


2. dock 中添加空白位置:

$ kill Dock, ‘ D ’ 要大写。

建议取消在dock设置中取消Mignificant 可以无限添加， 删除只需手动拖出dock, 或长按选择删除 — (见下图)。


3. Dock 动画提速 (提速包括其他窗口动画):

# 快
defaults write com.apple.dock autohide-time-modifier -int 0;killall Dock
# 超快
defaults write com.apple.dock autohide-time-modifier -float 0.12;killall Dock
# 还原
defaults delete com.apple.dock autohide-time-modifier;killall Dock


4. 播放(Star War)星战:

telnet towel.blinkenlights.nl


5. 修改默认截图前缀:

defaults write com.apple.screencapture name “自定义项目”
