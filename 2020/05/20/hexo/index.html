    Hexo

    安装

    $ npm install -g hexo-cli

    版本

    $ hexo -v

    建站

    $ mkdir ~/desktop/sites 文件夹名称

    $ cd ~/desktop/sites 文件夹名称

    $ hexo init

    $ npm install -g npm

    如果弹出： low severity vulnerability, npm 只是告诉你有弱点， 不用担心没有, not possible to be 100% Secure.

    $ nano ~/desktop/sites/_config.yml

    编辑： title , etc.,

    启动

    $hexo -s

    阅览器打开查看: http://localhost:4000

    写文章

    $ hexo new “文章名”

    日常清理

    $ hexo clean

    生成静态文件

    $ hexo g

    部署到 Gitgub

    $ npm install hexo-deployer-git –save

    $ nano ~/desktop/sites/_config.yml

    设置修改： _config.yml 中的 #deployment 章节

    # Deployment
    ## Docs: https://hexo.io/docs/deployment.html
    deploy:
    type: git
    repo: git@github.com: git用户名/仓库名.io.git
    # 默认master分支
    # 不建议加 message

    local search

    $ npm install hexo-generator-search –save

    让主题支持mermaid流程图:

    $ npm install hexo-filter-mermaid-diagrams

    修改 footer.ejs 文件

    <% if (theme.mermaid.enable) { %> <% } %>
    <%= __('footer.copyright') %> © <% var endyear="(theme.copyright" && theme.copyright.end\_year) ? theme.copyright.end\_year : new date().getfullyear() %> <% var startyear="(theme.copyright" && theme.copyright.start\_year) ? theme.copyright.start\_year : new date().getfullyear() %> <%= startyear>= endYear ? endYear : startYear + "-" + endYear %> <%= config.author || config.title %>
      <% for (var i in theme.nav) { %>
    • <%= \_\_('nav.'+i).replace("nav.", "") %>
      • <% } %>

    站点计数插件(hexo-wordcount)

    leancloud.cn 实现 hexo 阅读次数统计， 参考:

    $ npm install hexo-wordcount –save

    解决图片路径问题

    $ npm install https://github.com/CodeFalling/hexo-asset-image –save

    PDF 嵌入 hexo-pdf

    $ npm install –save hexo-pdf

    Migrate Wordprss site to Hexo

    安装迁移插件 hexo-migrator-wordpress

    $ npm install hexo-migrator-wordpress –save

    HTTPS

    HTTPS 强制不能勾选:

    因为当前页面中有非https的链接导致，比如图片图床不是https的，或者评论插件不支持https，不过我博客中使用的来必力评论是支持https，只是我当时的图床无都是http所以只能是显示叹号了，后来把所有图片图床换成https的就好啦。
