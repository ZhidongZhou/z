Coin-Row Problem 币值最大化问题

V1,V2…Vn

S(i-2) + Vi

As recurrence relation: S(i) = max{S(i-1), S(i-2) + Vi}

Base Case: S(0) = 0 and S(1) = V1

Naive Fibonacci: Lots of repetition of identical sub-computations.

function COINROW(C[1..𝓃])
    S[0] ← 0
    S[1] ← C[1]
    for i ← 2 to 𝓃
        S[i] ← max{S[i-1], S[i-2] + C[i]}
    return S[𝓃]
