NAATI & CCL 澳洲翻译课程

相·遇
08-05-2020
Hits
学习笔记 › CCL
CCL

NAATI 是一个翻译认证机构， 而CCL则是这个机构颁发的一个资格认证。

(NAATI 是 CCL 的‘爸爸’）



NAATI 澳大利亚全国翻译认证机构 —— (National Accreditation Authority for Translators and Interpreters).


改革：自30/JUN/2018，NAATI 取消了“内部考试”。


🙇🏻‍♂️ 交钱去读NAATI认证的校内课程（approved course）已经不能被作为取得认证的 ‘捷径’了。



所以CCL自2018年就诞生喽。


👉🏻 参加外部考试的人数是4333人，获得认证的有1255人，通过率仅29%

👉🏻 参加内部考试的人数是5431人，获得认证的有5155人，通过率达95%



—— 2016-2017 Financial Year



以下测试均由NAATI提供， 可用来获取5分移民加分， 其中CCL是最简单的。



CCL 社区语言资格认证 —— (Credentialed Community Language).

只有口译没有笔译

CCL考试只有口译， 并且有别与二级口译（certified provisional interpreter）& 三级口译(certified interpreter）， CCL 仅涵盖二级口译中的核心 (实务) 部分。



CCL Test v Certification Test

  • 翻译认证 \ Certification Test: 可以从事NAATI翻译工作 + 5分 CCL Point
  • 社区语言资格认证 \ CCL Test: 5分 CCL Point ONLY

NAATI CCL Test 社区语言资格认证 NAATI Certification Test 翻译认证
报考前提 NULL 不需要上课 必须参加：NAATI 认可的培训 + 职业道德考试 + 文化考试
考试形式 2篇录音 一篇电话翻译 + 两篇真是场景翻译
结果认证 快： 4-8周 通过提交即刻获取5分 慢： 3个月左右

CCL 就是用来获取 ⇢ 5分移民加分的 不可以用来从事翻译工作。


二者均属于： ‘外部’ 考试。
