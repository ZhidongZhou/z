Hits
IT › Terminal
MacOS


1. Finder中显示当前路径:

首先打开 Terminal

关闭路径显示， 只需改 YES 为 NO。


2. Dock 中添加空白位置

$ kill Dock, ‘ D ’ 要大写。

建议取消在dock设置中取消Mignificant 可以无限添加， 删除只需手动拖出dock, 或长按选择删除 — (见下图)。


3. Dock 动画提速 (提速包括其他窗口动画)


4. 使不活动的图标进入半透明状态：

defaults write com.apple.dock showhidden -bool TRUE; killall Dock


关闭:

defaults write com.apple.dock showhidden -bool False; killall Dock

Bonus. 播放(Star War)星战

telnet towel.blinkenlights.nl


5. 修改默认截图前缀:

defaults write com.apple.screencapture name "自定义项目"
