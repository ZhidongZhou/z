命令行玩转格式转换

相·遇
07-06-2020
Hits
IT › Terminal
MacOS

多格式通用 .png .jpg .jpeg & .icon, etc,.

修改图片像素大小

convert 文件名.jpg -resize 380x278 新文件名.jpg

转为黑白图片

convert 文件名.jpg -monochrome 新文件名.jpg

转为灰色图片

convert 文件名.jpg -colorspace Gray 新文件名.jpg

转为4张不同灰色图片

convert 文件名.png -separate 新文件名.jpg

转 .pdf 为 .jpg

正常.jpg
sips -s format jpeg 文件名.pdf --out 新文件名.jpg

透明背景

sips -s format png 文件名.pdf --out 新文件名.png

Batch Processing 将所有 .png 转为: .jpg:

for i in *.png ; do convert "$i" "${i%.*}.jpg" ; done

OR

ls -1 *.png | xargs -n 1 bash -c 'convert "$0" "${0%.*}.jpg"'

OR

mogrify -format jpg *.png

OR 同时删除原文件

mogrify -format jpg *.png && rm *.png

OR 同时删除原文件

find . -iname '*.png' | while read i; do mogrify -format jpg "$i" && rm "$i"; echo "Converted $i to ${i%.*}.jpg"; done

OR 同时删除原文件

for i in *.png; do mogrify -format jpg "$i" && rm "$i"; echo "$i converted to ${i%.*}.jpg"; done
